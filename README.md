# Tranzystor
Symulator tranzystora bipolarnego - projekt niezbędny by zaliczyć przedmiot Programowanie Obiektowe.
Więcej szczegółów w pliku Specyfikacja.pdf.

Aktualna wersja: RC 1.2.2 (release candidate)
